notes:

type python -m venv venv to create a virtual environment

to activate the virtual environment, type venv\Scripts\Activate.ps1 in powershell or venv\Scripts\activate.bat in the commmand prompt

or if using vscode workspace:

Open VS Code in your project folder.
Open Command Palette (Ctrl+Shift+P or Cmd+Shift+P).
Search and select: Python: Select Interpreter
Choose the interpreter inside your venv folder. It usually shows up as something like:
./venv/bin/python (macOS/Linux)
.\venv\Scripts\python.exe (Windows)
